# Title

If you want to add a title and description on the header of the chart, you can simply pass the following props:

- `title`: Title of the chart. (Can be either text/node)
- `titleDesc`: Description underneath the title. (Can be either text/node)
- `titleAlign`: Text align of the title. (`left`|`center`|`right`; `left` if it is not provided)

Charts in `@ichef/transcharts-chart` simply pass these props to [HeaderBox](/graph_components/header_box) of `@ichef/transcharts-graph` to render the title and description.


## Example

<Playground>
  <div style={{ width: '100%', height: '380px' }}>
    <LineChart
      title="Taiwan's GDP Per Capita"
      titleDesc={<span>Source: <a href="https://www.gapminder.org/" target="_blank">GapMinder</a> | Unit: constant 2011 international $</span>}
      titleAlign="right"
      data={taiwanData}
      x={{ field: 'year', type: 'ordinal' }}
      y={{ field: 'gdpPercap', type: 'quantitative' }}
    />
  </div>
</Playground>
